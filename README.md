## About the preview

mPDF is in **public preview**. Preview builds are free to use and each build
is valid for **45 days** from its release date (the exact date is shown in
every release's notes).

When a build reaches the end of its window it switches to **read-only mode**:

- ✅ Reading, searching, and exporting keep working
- ✅ **Save As** always works — your changes are never locked in
- ⛔ Editing, annotating, organizing pages, and in-place saving are paused

Download the newest preview from the
[Releases page](https://github.com/tacshi/mPDF-Release/releases/latest) to
pick up right where you left off. The app also shows an in-app notice when a
newer preview is available.

## Install

1. Download the `.dmg` from the
   [latest release](https://github.com/tacshi/mPDF-Release/releases/latest).
2. Open it and drag **mPDF** into **Applications**.
3. Launch mPDF from Applications.

**Requirements:** macOS on Apple Silicon or Intel (universal binary).

## Highlights

- **Reader-first design** — immersive reading with auto-hiding chrome,
  tabs, bookmarks, and full-text search
- **True content editing** — edit text in place, insert/replace/extract
  images, remove hidden text, redact
- **Annotations** — highlights, shapes, notes, and redaction marks
- **Page organizer** — reorder, rotate, insert, extract, split, and merge
  pages
- **Privacy tools** — find & redact, remove hidden text, strip web links,
  add/remove password protection
- **Housekeeping** — reduce file size, attachments, links panel

## Feedback

Found a bug or have a suggestion? Please
[open an issue](https://github.com/tacshi/mPDF-Release/issues) — feedback
during the preview directly shapes the 1.0 release.

When reporting a bug, it helps to include:

- The preview version (mPDF → About mPDF, or the DMG file name)
- Your macOS version
- Steps to reproduce, and a sample PDF if the issue is file-specific
  (only share files you're comfortable making public)

## Roadmap

The preview phase focuses on stability and feedback. After 1.0, mPDF will
be a paid app — preview users will have a smooth upgrade path.
